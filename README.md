# FirstRaytrace

## これは何??

[レイトレ合宿2!!](https://sites.google.com/site/raytracingcamp2/)への応募作品です。

+ ごく単純なパストレース
+ 食い違い量列を利用した準モンテカルロ法を実装
+ 「薄いレンズのカメラ」による被写界深度を実装

## ビルド環境

+ Xcode 5.1.1
+ VisualStudio2013

依存ライブラリはすべて同梱されています。
